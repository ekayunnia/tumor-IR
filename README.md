

**TumorCBIR**

-------------------------------------------------------------------------------------------------------------

**Deskripsi**
Sumber Dataset : https://figshare.com/articles/dataset/brain_tumor_dataset/1512427
Dataset Brain Tumor berjumlah 3064 image yang terbagi menjadi beberapa kelas jenis tumor yaitu :
- 1426 image Glioma,
- 708 image Meningioma,
- 930 image Pituitary
